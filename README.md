# CodeCraft

CodeCraft is a low-code/no-code platform designed to streamline the process of creating and managing workflows. It allows users to create, update, and execute workflows using predefined tools, simplifying automation for various tasks.

## Features

- **Create Workflow**: Users can create workflows from scratch or based on templates.
- **Add Tasks**: Easily add tasks to workflows and start them.
- **Execute Workflows**: Start workflows, track their status, and view execution logs.
- **Retry Execution**: Retry workflows in case of failure.
- **Track Status**: Monitor the status of workflows and individual tasks.
